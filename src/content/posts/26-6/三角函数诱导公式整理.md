# 三角函数诱导公式整理

## 1. 基本结论

例如：

```math
\sin(x+\pi)=-\sin x
```

这类公式叫做**三角函数诱导公式**，主要用于把复杂角度的三角函数化成基本角 `x` 的三角函数。

---

## 2. 加减 `π` 的公式

```math
\sin(x+\pi)=-\sin x
```

```math
\sin(x-\pi)=-\sin x
```

```math
\cos(x+\pi)=-\cos x
```

```math
\cos(x-\pi)=-\cos x
```

```math
\tan(x+\pi)=\tan x
```

```math
\tan(x-\pi)=\tan x
```

---

## 3. 加减 `2π` 的公式

```math
\sin(x+2\pi)=\sin x
```

```math
\cos(x+2\pi)=\cos x
```

```math
\tan(x+2\pi)=\tan x
```

周期规律：

```math
\sin x,\cos x \text{ 的周期是 } 2\pi
```

```math
\tan x \text{ 的周期是 } \pi
```

---

## 4. 加减 `π/2` 的公式

```math
\sin\left(x+\frac{\pi}{2}\right)=\cos x
```

```math
\sin\left(x-\frac{\pi}{2}\right)=-\cos x
```

```math
\cos\left(x+\frac{\pi}{2}\right)=-\sin x
```

```math
\cos\left(x-\frac{\pi}{2}\right)=\sin x
```

```math
\tan\left(x+\frac{\pi}{2}\right)=-\cot x
```

```math
\tan\left(x-\frac{\pi}{2}\right)=-\cot x
```

---

## 5. 负角公式

```math
\sin(-x)=-\sin x
```

```math
\cos(-x)=\cos x
```

```math
\tan(-x)=-\tan x
```

规律：

```math
\sin x,\tan x \text{ 是奇函数}
```

```math
\cos x \text{ 是偶函数}
```

---

## 6. `π` 与 `x` 的组合

```math
\sin(\pi-x)=\sin x
```

```math
\sin(\pi+x)=-\sin x
```

```math
\cos(\pi-x)=-\cos x
```

```math
\cos(\pi+x)=-\cos x
```

```math
\tan(\pi-x)=-\tan x
```

```math
\tan(\pi+x)=\tan x
```

---

## 7. `π/2` 与 `x` 的组合

```math
\sin\left(\frac{\pi}{2}-x\right)=\cos x
```

```math
\sin\left(\frac{\pi}{2}+x\right)=\cos x
```

```math
\cos\left(\frac{\pi}{2}-x\right)=\sin x
```

```math
\cos\left(\frac{\pi}{2}+x\right)=-\sin x
```

```math
\tan\left(\frac{\pi}{2}-x\right)=\cot x
```

```math
\tan\left(\frac{\pi}{2}+x\right)=-\cot x
```

## 9. 记忆口诀

三角函数诱导公式常用口诀：

> 奇变偶不变，符号看象限。

解释如下：

1. **奇变偶不变**

当角中含有：

```math
\frac{\pi}{2}\pm x,\frac{3\pi}{2}\pm x
```

这种形式时，函数名会发生变化：

```math
\sin \leftrightarrow \cos
```

```math
\tan \leftrightarrow \cot
```

当角中含有：

```math
\pi\pm x,2\pi\pm x
```

这种形式时，函数名不变。

2. **符号看象限**

先把原来的三角函数看作正角，然后判断该角所在象限，再决定结果前面的正负号。

## 11. 总结

最常用的转换规律可以概括为：

```math
\sin(x+\pi)=-\sin x
```

```math
\cos(x+\pi)=-\cos x
```

```math
\tan(x+\pi)=\tan x
```

```math
\sin\left(\frac{\pi}{2}+x\right)=\cos x
```

```math
\cos\left(\frac{\pi}{2}+x\right)=-\sin x
```

这些公式在化简三角函数、解三角方程、证明恒等式时非常常用。
